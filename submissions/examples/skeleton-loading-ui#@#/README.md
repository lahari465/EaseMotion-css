# Skeleton Loading UI

A modern, accessible skeleton loading state for content cards, built entirely with **EaseMotion CSS**. It provides visual feedback during data fetching, improving perceived performance and user experience.

## ✨ Features

- **Zero JavaScript**: Pure CSS implementation using native EaseMotion utility classes.
- **Native Pulse Animation**: Leverages EaseMotion's `.ease-pulse` class for a smooth, hardware-accelerated shimmer effect.
- **Staggered Entrance**: Uses `.ease-fade-in` and `.ease-delay-*` classes to make the skeleton itself load gracefully.
- **Highly Customizable**: Easy to adapt for articles, products, or user profiles by adjusting the CSS modifier classes.
