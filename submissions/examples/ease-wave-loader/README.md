# 📊 Wave Loader – Bars Wave Like Equalizer

> A dynamic, rhythmic loading animation where bars wave like a music equalizer, creating a vibrant, energetic visual experience perfect for audio-related interfaces.

---

## 📖 Description

The **Wave Loader** component creates a dynamic, rhythmic loading animation where ten bars wave like a music equalizer. Each bar animates with a staggered delay, creating a cascading wave effect that mimics the visual patterns of an audio equalizer. The bars have varying heights, gradient colors, and a smooth bounce animation that feels alive and energetic.

The component uses CSS keyframe animations with `transform: scaleY()` to create the wave effect. Each bar has a different `animation-delay` (0.08s increments) for a cascading wave that flows from left to right. The bars feature gradient colors and opacity variations, adding visual depth and interest. The animation is smooth, responsive, and performance-optimized.

Perfect for music players, audio applications, loading states, and any interface where you want to create a vibrant, energetic visual experience. The Wave Loader transforms a simple loading indicator into a dynamic, engaging visual that feels alive.
