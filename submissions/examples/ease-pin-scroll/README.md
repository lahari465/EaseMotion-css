# 📌 Pin Scroll – Sections Snap into View

> A smooth, presentation-style scrolling experience where sections snap into view as you scroll, creating a polished, guided content journey.

---

## 📖 Description

The **Pin Scroll** component creates a smooth, presentation-style scrolling experience where sections snap into view as you scroll. Each section locks into place, creating a polished, guided journey through your content. The component features full-page sections with smooth snap scrolling, progress indicators, keyboard navigation, and active state tracking.

The component uses `scroll-snap-type: y mandatory` for native snap scrolling, with JavaScript enhancements for active section tracking, progress indicators, and keyboard navigation. Each section has a unique gradient background and animated content that fades and slides in when active.

Perfect for landing pages, presentations, portfolios, and any application where you want to create a guided, immersive content experience. The Pin Scroll transforms a simple scroll into a polished, presentation-like journey.
