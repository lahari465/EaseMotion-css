# Shimmer Sweep Popover

A responsive shimmer sweep popover built using only HTML and CSS.

## Features

- Pure HTML & CSS
- Shimmer sweep animation
- Responsive layout
- Accessible hover/focus interaction
- prefers-reduced-motion support
- No JavaScript

## Folder Structure

```
shimmer-sweep-popover/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in your browser and hover or focus on the button to display the shimmer popover.
