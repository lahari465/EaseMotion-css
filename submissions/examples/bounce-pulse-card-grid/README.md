# CSS Bounce Pulse Card Grid

A responsive card grid featuring a bounce-pulse hover animation built using only HTML and CSS.

## Features

- Pure HTML & CSS
- Bounce pulse hover animation
- Responsive grid layout
- CSS variables for customization
- Accessibility with prefers-reduced-motion

## Usage

```html
<section class="cards">
    <div class="card">
        <h2>Portfolio</h2>
        <p>Project description</p>
    </div>
</section>
```

## CSS Variables

```css
:root{
    --primary:#5b5df5;
    --bg:#f5f7fb;
    --card:#ffffff;
}
```

## Accessibility

Supports:

- prefers-reduced-motion
- Responsive layouts
- Keyboard-friendly HTML structure
