# Glitch-Flicker Badge Component

A lightweight, pure CSS badge module featuring cybernetic RGB split glitch impulses and chromatic flicker effects.

## Features
- **Pure CSS/HTML**: Animated via CSS pseudo-elements and clip-path keyframe trajectories with zero external JavaScript dependencies.
- **Accessible**: Built with native support for `prefers-reduced-motion` media queries.
- **Customizable**: Exposes CSS custom properties to adjust glitch frequency, RGB displacement offsets, and colors.

## Customization Parameters
The following CSS custom properties are available in `style.css`:
- `--glitch-duration`: Controls the frequency interval of periodic glitch-flicker passes.
- `--glitch-offset-x`: Horizontal pixel displacement offset during chromatic splitting.
- `--glitch-offset-y`: Vertical pixel displacement offset during chromatic splitting.
