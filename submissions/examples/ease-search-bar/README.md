# 🔍 Search Bar – Expands with Focus

> A clean, modern search bar that smoothly expands when focused, with a clear button, search suggestions, and a polished, user-friendly design.

---

## 📖 Description

The **Search Bar** component creates a clean, modern search bar that smoothly expands when focused. The search bar widens with a satisfying animation, the search icon changes color, and a clear button appears when text is entered. The component includes search suggestions that slide down when the search bar is expanded, providing a complete search experience.

The component uses CSS transitions with `cubic-bezier` easing for smooth expansion, and JavaScript for handling focus, input, clear, and suggestion interactions. The search suggestions provide contextual feedback, updating based on the user's input. The clear button allows users to quickly reset the search field.

Perfect for navigation headers, landing pages, blogs, and any application where you need a polished, user-friendly search experience. The Search Bar transforms a simple input into a delightful, interactive component.
