# Float Banner Example

Resolves #54675

An announcement/promotion banner component designed to hover and float with a smooth bobbing animation cycle.

## What does this do?
Draws attention to a top-level alert/banner by continuously translating it vertically (`translateY`) in a slow, ease-in-out breathing rhythm.

## How is it used?
```html
<div class="float-banner-rb" role="banner">
  <span>Promotion details</span>
</div>
```

## Why is it useful?
The ambient floating motion makes the banner stand out from static grid elements on page load, driving higher conversion rates on active alerts.
