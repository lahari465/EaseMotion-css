How It Works

Each tooltip uses a hidden checkbox toggle. When checked, the tooltip slides up with a translate transform and elastic cubic-bezier easing. The arrow is a pure CSS border trick.

Accessibility

Tooltips include role="tooltip" for screen readers. The prefers-reduced-motion media query disables all transitions for users who prefer reduced motion.
