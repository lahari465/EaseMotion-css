# Ease Kubernetes Resource Quota Gauge

## Overview

A responsive Kubernetes resource management component for visualizing namespace quota consumption.

## Features

- Circular CPU quota gauge
- Memory usage visualization
- Pod quota tracking
- Storage quota monitoring
- Service and PVC limits
- Quota threshold indicators
- Critical storage animation
- Namespace quota information
- Responsive layout
- Hover interactions
- Pure HTML and CSS

## Folder Structure

- `demo.html`
- `style.css`
- `README.md`

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

This component provides a reusable Kubernetes resource management interface for understanding namespace quota consumption through animated gauges, progress indicators, and threshold warnings.
