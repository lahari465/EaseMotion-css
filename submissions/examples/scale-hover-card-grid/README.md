# CSS Scale Hover Card Grid

A responsive card grid featuring a smooth scale-hover animation using only HTML and CSS.

## Features

- Pure HTML & CSS
- Smooth scale hover animation
- Responsive grid layout
- CSS custom properties
- Accessible with `prefers-reduced-motion`

## Usage

```html
<section class="card-grid">
    <div class="card">
        <h2>Portfolio</h2>
        <p>Project description</p>
    </div>
</section>
```

## CSS Variables

```css
:root{
    --primary:#4f46e5;
    --background:#f5f7fb;
    --card:#ffffff;
    --duration:.35s;
}
```

## Accessibility

- Responsive design
- Keyboard-friendly HTML
- Supports `prefers-reduced-motion`
