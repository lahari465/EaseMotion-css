# 🔲 Fade Grid – Squares Fade In/Out in Grid

> A mesmerizing, hypnotic animation where squares in a grid pattern fade in and out with staggered delays, creating a captivating visual experience.

---

## 📖 Description

The **Fade Grid** component creates a mesmerizing, hypnotic animation where nine squares in a 3x3 grid pattern fade in and out with staggered delays. Each square pulses with a smooth opacity and scale animation, creating a wave-like effect that flows across the grid. The squares have gradient colors, adding visual depth and interest.

The component uses CSS keyframe animations with `opacity` and `transform: scale()` to create the fade pulse effect. Each cell has a different `animation-delay` (0.15s increments) for a cascading wave that ripples through the grid. The cells fade from low opacity to full opacity and back, with subtle scaling for a breathing effect.

Perfect for loading states, background animations, and any interface where you want to create a mesmerizing, dynamic visual experience. The Fade Grid transforms a simple grid into a captivating, hypnotic display.
