# Glow Loader (Warning State)

A lightweight CSS loader that displays a smooth glowing warning animation using only HTML and CSS.

---

## Overview

This example demonstrates a reusable warning-style loading indicator with a glowing rotating ring and animated center pulse.

The loader is designed to attract attention without being distracting, making it suitable for loading states that require additional visual emphasis.

---

## Features

- ✨ Smooth rotating loader animation
- 💛 Warning-themed yellow glow
- 🔄 Infinite rotation
- 🌟 Animated glowing center pulse
- ⚡ Pure CSS implementation
- ♿ Supports `prefers-reduced-motion`
- 📱 Responsive design
- 🎯 Beginner-friendly code structure

## How to Use

1. Copy the HTML markup into your project.
2. Include the `style.css` file.
3. Customize the colors or animation speed if desired.

Example:

```html
<div class="glow-loader-example-ag">
    <div class="loader-ring"></div>
    <div class="loader-core"></div>
</div>
```

## Animation Details

The loader combines multiple CSS animations to create a polished loading effect.

### Loader Ring

- Rotates continuously
- Emits a soft glowing pulse
- Uses a warning yellow accent

### Loader Core

- Pulses smoothly
- Expands and contracts gently
- Adds depth to the animation
