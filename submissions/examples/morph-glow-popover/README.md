# CSS Morph-Glow Popover

A modern popover component with a morphing scale animation and glow effect built using pure HTML and CSS.

## Features

- Pure HTML & CSS
- Morph + Glow animation
- Responsive layout
- CSS Custom Properties
- Keyboard accessible
- Supports `prefers-reduced-motion`
