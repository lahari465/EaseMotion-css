How It Works

Radio buttons control which slide is visible. The carousel track uses CSS transform translateX to slide between items with elastic easing. Navigation dots and arrows are labels tied to the radio inputs.

Accessibility

Carousel uses role="region", aria-roledescription="slide", and role="tablist" for screen readers. The prefers-reduced-motion media query disables all slide transitions.
