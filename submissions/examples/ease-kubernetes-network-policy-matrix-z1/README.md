# Ease Kubernetes Network Policy Matrix

## Overview

A responsive Kubernetes network security component for visualizing communication permissions between namespaces and workloads.

## Features

- Network policy matrix
- Allowed traffic states
- Denied traffic states
- Restricted communication indicators
- Animated active traffic state
- Namespace communication overview
- Policy rule details
- Security summary metrics
- Hover interactions
- Responsive behavior
- Pure HTML and CSS

## Folder Structure

- `demo.html`
- `style.css`
- `README.md`

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

The component provides a reusable Kubernetes security visualization for understanding network isolation and communication rules through a compact access-control matrix.
