# Glitch Flicker Stepper

A responsive multi-step progress indicator built with pure HTML and CSS. The active step features a subtle glitch flicker animation inspired by cyber-style interfaces.

## Features

- Pure HTML5 & CSS3
- Responsive desktop, tablet, and mobile layout
- Smooth glitch flicker animation
- CSS custom properties for easy customization
- No JavaScript required
- Supports `prefers-reduced-motion`

## CSS Custom Properties

```css
--primary
--completed
--inactive
--background
--surface
--text
--step-size
--animation-speed
```

## Accessibility

Users who enable reduced motion in their operating system automatically receive a static version with animations disabled.

## Browser Support

Works in all modern browsers supporting CSS animations and media queries.
