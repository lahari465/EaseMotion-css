# ⌨️ Typewriter Effect – Types Character by Character

> A realistic typewriter effect that types text character by character with adjustable speed, pause/resume functionality, and keyboard shortcuts.

---

## 📖 Description

The **Typewriter Effect** component creates a realistic typewriter effect that types text character by character with adjustable speed. The component features a dark terminal-style display, blinking cursor, and controls for starting, pausing, and resetting the typing. The speed can be adjusted in real-time using a slider, and keyboard shortcuts provide quick access to controls.

The component uses JavaScript with `setTimeout` and random delays (70-130% of base speed) for realistic typing rhythm. Multiple lines of text can be typed sequentially, and the cursor blinks when idle. The design is clean and modern, with a dark terminal aesthetic.

Perfect for landing pages, hero sections, developer portfolios, and any application where you want to create a dramatic, engaging text reveal. The Typewriter Effect transforms a simple text display into a captivating, interactive experience.
