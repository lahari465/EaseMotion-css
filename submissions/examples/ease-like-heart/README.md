# ❤️ Like Heart – Bursts into Particles

> A delightful, interactive like button where the heart bursts into colorful particles on click, creating a celebratory, engaging experience.

---

## 📖 Description

The **Like Heart** component creates a delightful, interactive like button where the heart bursts into colorful particles on click. When clicked, the heart icon pops with a satisfying animation, colorful particles explode outward, and the like count increments. The effect creates a celebratory, engaging experience that makes liking feel fun and rewarding.

The component uses JavaScript to create and animate particles with random colors, sizes, and trajectories. Each click generates 40 particles that fly outward in random directions with varying velocities. The heart button animates with a pop effect, and the like count updates in real-time. The particles fade out and are removed after the animation completes.

Perfect for social media apps, content platforms, and any application where you want to make the "like" interaction feel special and engaging.
