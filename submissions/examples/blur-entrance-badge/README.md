# CSS Blur Entrance Badge

A responsive badge component featuring a smooth blur entrance animation using only HTML and CSS.

## Features

- Pure HTML & CSS
- Blur Entrance Animation
- Hover Lift Effect
- Fully Responsive
- Accessible
- Supports `prefers-reduced-motion`
- No JavaScript

## Folder Structure

```text
blur-entrance-badge/
├── demo.html
├── style.css
└── README.md
```

## Usage

```html
<link rel="stylesheet" href="style.css">

<span class="badge">NEW</span>
```

## Customization

You can customize:

- Background color
- Text color
- Animation duration
- Animation delay
- Border radius
- Shadow
- Padding

inside `style.css`.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Accessibility

- Responsive layout
- Reduced motion support
