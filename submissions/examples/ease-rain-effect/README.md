# 🌧️ Rain Effect – Drops Fall on Canvas

> A realistic, immersive rain effect where raindrops fall on a canvas, creating a calming, atmospheric weather visualization with interactive controls.

---

## 📖 Description

The **Rain Effect** component creates a realistic, immersive rain visualization where raindrops fall on a canvas with varying speeds, lengths, and opacities. The effect features a dark gradient background, realistic raindrops with wind drift, splash effects on the ground, and a subtle mist at the bottom. The component includes interactive controls for pausing/resuming, adjusting drop density (20-200 drops), and resetting the effect.

The component uses HTML Canvas with JavaScript animation loops (`requestAnimationFrame`) for smooth 60fps performance. Each raindrop has unique properties (speed, length, opacity, width, wind drift), creating a natural, organic rain effect. Splash particles appear when drops reach the bottom, adding to the realism.

Perfect for ambient backgrounds, relaxation apps, meditation tools, and any application where you want to create a calming, atmospheric weather effect. The Rain Effect transforms a blank canvas into a soothing, immersive experience.
