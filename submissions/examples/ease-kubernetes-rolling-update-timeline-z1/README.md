# Ease Kubernetes Rolling Update Timeline

## Overview

A responsive Kubernetes deployment component for visualizing rolling update progress and revision lifecycle events.

## Features

- Deployment rollout timeline
- Old and new ReplicaSet visualization
- Updated pod indicators
- Rollout progress animation
- Readiness verification state
- Pending replica termination state
- Revision information
- RollingUpdate strategy details
- Hover and pulse animations
- Responsive layout
- Pure HTML and CSS

## Folder Structure

- `demo.html`
- `style.css`
- `README.md`

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

The component provides a reusable Kubernetes deployment visualization for understanding rolling update progress, ReplicaSet transitions, pod readiness, and rollout lifecycle events.
