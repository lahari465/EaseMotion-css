# ⚪ Bouncing Balls – Three Balls Bounce in Sync

> A dynamic, energetic loading animation where three colorful balls bounce in perfect sync with a satisfying rhythm and elastic motion.

---

## 📖 Description

The **Bouncing Balls** component creates a dynamic, energetic loading animation where three colorful balls bounce in perfect sync with a satisfying, elastic rhythm. Each ball has a unique color and staggered delay, creating a cascading wave-like effect. The balls squash and stretch as they bounce, mimicking realistic physics and adding a sense of weight and life to the animation.

The component uses CSS keyframe animations with `transform` and `scale` to create the bouncing effect. Each ball has a staggered `animation-delay` (0.1s increments) for a cascading wave effect. Shadows beneath the balls pulse in sync with the bounce, adding depth and realism. The balls squash at the bottom and stretch at the top, mimicking realistic elastic physics.

Perfect for loading states, waiting screens, and any interactive element where you want to indicate activity with a playful, energetic animation. The Bouncing Balls transform a simple loading indicator into a fun, engaging visual experience.
