# Ease Kubernetes Namespace Resource Quota

## Overview

A responsive Kubernetes operations dashboard for visualizing namespace resource quota consumption.

## Features

- CPU quota monitoring
- Memory utilization tracking
- Pod count limits
- Persistent storage monitoring
- Animated resource progress bars
- Quota threshold warnings
- Namespace metadata
- Interactive hover states
- Responsive dashboard layout
- Pure HTML and CSS

## Folder Structure

- `demo.html`
- `style.css`
- `README.md`

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

This component provides a reusable Kubernetes resource monitoring interface that makes namespace quota consumption and capacity risks easy to understand through animated visual indicators.
