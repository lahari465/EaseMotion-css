# Responsive Statistics Cards

A lightweight and reusable HTML/CSS component for displaying important statistics in responsive cards.

## Features

- Fully responsive layout
- Modern card design
- Smooth hover animation
- Lightweight HTML & CSS only
- Accessible and easy to customize
- Supports reduced-motion preferences

## Folder Structure

```text
submissions/examples/feature-responsive-statistics-cards-ds/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` in any browser.

The component can be integrated into:

- Dashboards
- SaaS landing pages
- Admin panels
- Portfolio websites
- Analytics pages

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to reduce animations for users who prefer minimal motion.

## Customization

You can easily modify:

- Card width
- Colors
- Icons
- Statistics
- Labels
- Hover effects

## Why is it Useful?

Statistics cards provide a clean and engaging way to present important metrics while maintaining responsiveness and simplicity.
