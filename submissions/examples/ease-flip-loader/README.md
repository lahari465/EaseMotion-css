# 🃏 Flip Loader – Cards Flip Continuously

> A mesmerizing, dynamic loading animation where cards flip continuously in a staggered cycle, creating a captivating, casino-inspired visual experience.

---

## 📖 Description

The **Flip Loader** component creates a mesmerizing, dynamic loading animation where three cards flip continuously in a staggered cycle. Each card features a different color and icon on the front and back, creating a visually rich, engaging display. The cards flip with a smooth 3D rotation, using `perspective` and `transform-style: preserve-3d` for realistic depth.

The component uses CSS keyframe animations with `transform: rotateY()` to create the flipping effect. Each card has a different `animation-delay` (0.3s increments) for a staggered, cascading cycle. The cards feature gradient colors, unique icons, and a polished design reminiscent of playing cards.

Perfect for loading states, gaming interfaces, casino-themed applications, and any interface where you want to create a dynamic, engaging visual experience. The Flip Loader transforms a simple loading indicator into a captivating, playful display.
