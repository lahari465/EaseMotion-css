# Bounce Pulse Stepper

A lightweight animated stepper component built using pure HTML and CSS.

## Features

- Pure HTML5 + CSS
- Responsive layout
- Bounce + pulse active step animation
- Accessible
- Supports `prefers-reduced-motion`
- No JavaScript required

## CSS Custom Properties

```css
--primary
--completed
--inactive
--bg
--text
--size
--speed
```

## Folder Structure

bounce-pulse-stepper/

## Responsive

- Desktop
- Tablet
- Mobile

## Accessibility

The animation automatically disables when the user enables reduced motion.
