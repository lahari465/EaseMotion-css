# Ease Kubernetes CronJob Schedule Orbit

## Overview

A responsive Kubernetes workload automation component for visualizing CronJob schedules, executions, and recent job activity.

## Features

- Animated CronJob schedule orbit
- Upcoming execution countdown
- Cron expression display
- Running job state
- Successful execution state
- Delayed job warning
- Recent execution history
- Job duration information
- Rotating orbit animations
- Hover and pulse interactions
- Responsive layout
- Pure HTML and CSS

## Folder Structure

- `demo.html`
- `style.css`
- `README.md`

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

This component provides a reusable Kubernetes CronJob visualization for understanding scheduled workloads, upcoming executions, running jobs, and recent job history through an animated orbital interface.
