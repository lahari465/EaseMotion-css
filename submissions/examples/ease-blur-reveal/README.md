# 🌫️ Blur Reveal – Blurred Text Sharpens on Hover

> A dramatic, engaging visual effect where blurred text sharpens on hover, creating a captivating reveal experience.

---

## 📖 Description

The **Blur Reveal** component creates a dramatic, engaging visual effect where blurred text sharpens on hover. Each word starts blurred and slightly scaled down, and when you hover over the container, the words sharpen one by one with staggered delays, creating a wave-like reveal effect. The words also feature a subtle scale and opacity transition for added polish.

The component uses CSS `filter: blur()` and `transition` with staggered delays for each word. The effect works on individual words, allowing for creative text reveals. The example items below also feature a similar blur reveal effect on their headings.

Perfect for landing pages, hero sections, portfolios, and any application where you want to create a dramatic, engaging text reveal. The Blur Reveal transforms a simple text display into a captivating, interactive experience.
