# Ease Kubernetes Service Mesh Latency Map

## Overview

A responsive Kubernetes service-mesh component for visualizing microservice communication, traffic flow, and request latency.

## Features

- Microservice topology visualization
- Animated request packets
- Service-to-service traffic paths
- Request volume monitoring
- Average latency information
- Slow service path detection
- Latency threshold alerts
- Animated service health states
- Responsive topology layout
- Hover interactions
- Pure HTML and CSS

## Folder Structure

- `demo.html`
- `style.css`
- `README.md`

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

The component provides a reusable service-mesh visualization for understanding Kubernetes microservice communication, traffic flow, and latency through animated topology connections.
