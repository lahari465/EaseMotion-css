# Ease Kubernetes Horizontal Pod Autoscaler

## Overview

A responsive Kubernetes workload component for visualizing Horizontal Pod Autoscaler activity and replica scaling.

## Features

- Current and target replica visualization
- CPU and memory utilization monitoring
- HPA target indicators
- Scale-up animation
- Scale-down animation
- Stable workload state
- Minimum and maximum replica information
- Recent scaling event display
- Responsive layout
- Pure HTML and CSS

## Folder Structure

- `demo.html`
- `style.css`
- `README.md`

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

The component provides a reusable Kubernetes autoscaling visualization that makes replica changes, resource targets, and scaling activity easy to understand through animated workload states.
