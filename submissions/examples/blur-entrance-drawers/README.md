# CSS Blur Entrance Drawer

A responsive drawer component featuring a smooth blur-to-clear entrance animation built using pure HTML and CSS.

## Features

- Pure HTML & CSS
- Blur entrance animation
- Responsive drawer
- CSS Custom Properties
- Keyboard accessible
- Supports prefers-reduced-motion

```
README.md
```

## CSS Variables

```css
:root{
  --drawer-width:280px;
  --bg:#1f2937;
  --accent:#3b82f6;
  --text:#fff;
  --duration:.45s;
}
```
