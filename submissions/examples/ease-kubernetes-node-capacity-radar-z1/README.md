# Ease Kubernetes Node Capacity Radar

## Overview

A responsive Kubernetes infrastructure component for monitoring resource capacity across cluster nodes.

## Features

- Node-level CPU monitoring
- Memory utilization tracking
- Storage capacity visualization
- Running pod information
- Healthy, busy, pressure, and critical states
- Animated capacity meters
- Node health indicators
- Responsive node grid
- Hover interactions
- Pure HTML and CSS

## Folder Structure

- `demo.html`
- `style.css`
- `README.md`

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

The component provides a reusable infrastructure monitoring interface for visualizing Kubernetes node capacity and identifying resource pressure before workloads are affected.
