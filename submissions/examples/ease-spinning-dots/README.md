# 🔵 Spinning Dots – Dots Orbit in Circle

> A classic, mesmerizing loading animation where colorful dots orbit in a circle with pulsing motion, creating a dynamic, engaging visual experience.

---

## 📖 Description

The **Spinning Dots** component creates a classic, mesmerizing loading animation where twelve colorful dots orbit in a circle like planets around a sun. Each dot pulses in size and opacity as it orbits, creating a dynamic, organic feel that's both engaging and calming. The dots have varying sizes and colors, adding visual depth and interest.

The component uses CSS animations with `transform` and `keyframes` to create the orbital motion. The dots are positioned evenly around the circle (30° apart) using `rotate()` and `translateX()`. Each dot has a staggered `animation-delay` for a smooth, cascading pulse effect. The spinner rotates continuously, while each dot pulses independently.

Perfect for loading states, waiting screens, and any interactive element where you want to indicate activity with a polished, professional animation. The Spinning Dots transform a simple loading indicator into a captivating visual experience.
