# Vibrate Textarea Example

A lightweight HTML + CSS example demonstrating a textarea with a vibration animation.

## Features

- Pure HTML & CSS
- Smooth vibration animation
- Hover and focus interaction
- Responsive layout
- Accessible textarea
- Supports `prefers-reduced-motion`

## Files

```
demo.html
style.css
README.md
```

## Accessibility

- Uses semantic HTML.
- Includes an accessible `aria-label`.
- Animation is disabled when users prefer reduced motion.

## Preview

Hover or focus the textarea to trigger the vibration effect.
