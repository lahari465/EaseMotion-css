# Float Drift Feature Grid

A responsive feature grid built with pure HTML and CSS. Each feature card gently floats using a lightweight CSS animation to create a modern and engaging layout.

## Features

- Pure HTML5 & CSS3
- Floating drift animation
- Responsive CSS Grid layout
- Hover elevation effect
- CSS custom properties for easy customization
- No JavaScript required
- Supports `prefers-reduced-motion`

## CSS Custom Properties

```css
--primary
--background
--surface
--text
--muted
--shadow
--radius
--animation-speed
```

## Accessibility

Animations are automatically disabled when users enable **Reduce Motion** in their operating system, ensuring an accessible experience.

## Browser Support

Compatible with all modern browsers supporting CSS Grid, custom properties, animations, and media queries.
