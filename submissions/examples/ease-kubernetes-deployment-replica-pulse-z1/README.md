# Ease Kubernetes Deployment Replica Pulse

## Overview

A responsive Kubernetes workload monitoring component for visualizing Deployment replica availability and rollout health.

## Features

- Desired and ready replica overview
- Replica dot visualization
- Stable deployment state
- Animated rolling update state
- Degraded workload warnings
- Failed replica indicators
- Rollout progress bars
- Responsive deployment grid
- Hover and pulse animations
- Pure HTML and CSS

## Folder Structure

- `demo.html`
- `style.css`
- `README.md`

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

This component provides a reusable visual interface for understanding Kubernetes Deployment replica health and rollout progress without requiring external JavaScript or UI libraries.
