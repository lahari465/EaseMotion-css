# CSS Elastic Slide Navbar

A responsive navigation bar featuring an elastic slide hover effect using only HTML and CSS.

## Features

- Pure HTML & CSS
- Responsive layout
- Elastic hover animation
- Accessible with prefers-reduced-motion support
- Easy customization using CSS variables

## Usage

```html
<nav class="elastic-navbar">
    <a href="#" class="active">Home</a>
    <a href="#">Projects</a>
    <a href="#">About</a>
</nav>
```

## CSS Variables

```css
:root{
    --primary:#4f46e5;
    --bg:#0f172a;
    --text:#ffffff;
    --duration:.4s;
}
```

## Accessibility

- Keyboard friendly
- Responsive
- Supports `prefers-reduced-motion`
