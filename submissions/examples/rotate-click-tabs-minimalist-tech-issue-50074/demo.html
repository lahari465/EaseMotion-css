ease-rotate-click-tabs-minimal-tech

Pure CSS tabs — minimalist tech surface, rotate-click panel transitions, zero JavaScript.

Status

Tab, or use Arrow keys once focused, to move between tabs — the underlying <input type="radio"> group handles all of it natively, no JavaScript involved.

Logs

On click, the incoming panel snaps in with a quick rotate-and-settle — flat, mechanical, no bounce — matching a minimalist tech feel.

Config

Tune --ease-tabs-tech-duration, --ease-tabs-tech-easing, and --ease-tabs-tech-rotate to change the feel of the rotation.
