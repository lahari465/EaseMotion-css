# 🎨 Color Shift Scroll – Background Changes Color

> A smooth, immersive scrolling experience where the background transitions through a curated palette of colors as you scroll down the page.

---

## 📖 Description

The **Color Shift Scroll** component creates a smooth, immersive scrolling experience where the background transitions through a curated palette of colors as you scroll down the page. Each section has a target color, and the background smoothly interpolates between colors as you scroll, creating a seamless, dynamic visual experience.

The component uses JavaScript to calculate scroll progress and interpolate between colors using RGB values. Each section features a glass-morphism card with the color name, a color indicator, and descriptive text. The background transitions smoothly with `requestAnimationFrame` for 60fps performance.

Perfect for landing pages, portfolios, brand stories, and any application where you want to create a dynamic, immersive visual experience that evolves with the user's scroll.
