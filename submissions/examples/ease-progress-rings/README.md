# 🍩 Progress Rings – Donut Fills with Animation

> A smooth, circular progress indicator where donut-shaped rings fill with a satisfying animated stroke, perfect for showing progress, loading states, and completion metrics.

---

## 📖 Description

The **Progress Rings** component creates smooth, circular progress indicators where donut-shaped rings fill with a satisfying animated stroke. The rings use SVG circles with `stroke-dasharray` and `stroke-dashoffset` to create the fill effect, with a spring-like easing for a polished, professional feel.

The component features three rings with different progress values (75%, 45%, 90%), each with a unique color and label. The rings animate on load with a smooth transition and can be restarted by clicking. The labels update to show the progress percentage, and hover effects provide interactive feedback.

Perfect for dashboards, loading states, skill indicators, and any interface where you need to show progress or completion metrics in a visually appealing way. The Progress Rings transform a simple progress bar into a polished, professional visual experience.
