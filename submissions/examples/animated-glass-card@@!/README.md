# Animated Glassmorphism Card

A beautiful, responsive glassmorphism profile card component built with **EaseMotion CSS**. It showcases staggered entrance animations and smooth hover interactions without any JavaScript.

## ✨ Features

- **Zero JavaScript**: Pure CSS animations using native EaseMotion utility classes.
- **Glassmorphism Design**: Modern frosted glass effect using CSS `backdrop-filter`.
- **Staggered Entrance**: Elements slide up sequentially using `ease-slide-up` and `ease-delay-*` classes.
- **Interactive Hover**: Smooth lift effect on the card and grow effect on buttons using `ease-hover-lift` and `ease-hover-grow`.
- **Fully Responsive**: Adapts gracefully to all screen sizes.
