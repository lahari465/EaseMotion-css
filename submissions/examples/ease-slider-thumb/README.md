# 🎚️ Slider Thumb – Thumb Has Glow/Pulse

> A modern, interactive slider with a glowing, pulsing thumb that provides satisfying visual feedback and a polished user experience.

---

## 📖 Description

The **Slider Thumb** component creates a modern, interactive slider with a glowing, pulsing thumb. The thumb features a subtle pulse animation with a soft glow, and on hover and click, it scales and intensifies the glow effect. The slider includes a floating value display that follows the thumb position, providing real-time feedback.

The component uses CSS animations for the pulse effect and custom styling for the thumb across WebKit and Firefox browsers. The slider track has a gradient fill, and the value display updates dynamically as the user drags the thumb. Double-clicking resets the slider to the middle position.

Perfect for modern interfaces, dashboards, and any application where you want a polished, interactive slider experience. The glowing, pulsing thumb adds a touch of sophistication and engagement to a standard form control.
