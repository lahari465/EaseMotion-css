# Ease Kubernetes CronJob Schedule Monitor

## Overview

A responsive Kubernetes operations component for visualizing CronJob schedules, upcoming executions, successful jobs, delays, and failed scheduled workloads.

## Features

- CronJob schedule timeline
- Successful execution state
- Upcoming job indicator
- Delayed execution warning
- Failed job state
- Cron schedule metadata
- Runtime and retry information
- Animated live indicators
- Responsive layout
- Pure HTML and CSS

## Folder Structure

- `demo.html`
- `style.css`
- `README.md`

## Usage

Open `demo.html` in a modern web browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

This component provides a reusable schedule-oriented interface for visualizing Kubernetes CronJob activity while keeping scheduled workload health easy to understand.
