# CSS Glitch Flicker Badge

A pure HTML and CSS badge component featuring a futuristic glitch flicker animation suitable for modern dashboards, portfolios, and UI highlights.

## Features

- Pure HTML & CSS
- Responsive Layout
- Glitch Hover Animation
- Flicker Glow Effect
- No JavaScript
- Accessible
- Supports prefers-reduced-motion
- Lightweight

## Folder Structure

```
glitch-flicker-badge/
├── demo.html
├── style.css
└── README.md
```

## Usage

```html
<link rel="stylesheet" href="style.css">

<span class="badge" data-text="NEW">NEW</span>
```

## Customization

Modify:

- Badge colors
- Glow intensity
- Border radius
- Animation speed
- Font size
- Padding

inside `style.css`.
