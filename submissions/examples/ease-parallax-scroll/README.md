# 🌄 Parallax Scroll – Layers Move at Different Speeds

> A stunning, immersive parallax scrolling effect where multiple layers move at different speeds, creating a sense of depth and dimension that brings your content to life.

---

## 📖 Description

The **Parallax Scroll** component creates a stunning, immersive scrolling experience where four layers (background, mountains, trees, and foreground) move at different speeds as you scroll, creating a sense of depth and dimension. The effect uses `requestAnimationFrame` throttling for smooth 60fps performance and responsive design for all screen sizes.

The component features a dark, atmospheric landscape with gradient backgrounds, mountain silhouettes, tree icons, and a foreground with animated text. Each layer moves at a different speed (0.15x, 0.30x, 0.50x, 0.70x) to create the parallax effect, mimicking the natural depth perception of real-world scenery.

Perfect for landing pages, creative portfolios, storytelling websites, and any application where you want to create a visually stunning, immersive experience. The Parallax Scroll transforms a simple scroll into a cinematic journey through a layered landscape.
