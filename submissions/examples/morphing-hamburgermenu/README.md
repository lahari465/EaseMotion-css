# CSS-only Morphing Hamburger Menu

A modern hamburger menu button that smoothly transforms into a close (✕) icon using only HTML and CSS.

## Features

- Pure HTML & CSS
- No JavaScript required
- Smooth morphing animation
- Responsive design
- Easy to customize
- Lightweight and reusable

## Folder Structure

```
morphing-hamburger-menu/
├── demo.html
├── style.css
└── README.md
```

## How It Works

- A hidden checkbox controls the animation state.
- Three horizontal bars form the hamburger icon.
- When checked:
  - The top bar rotates 45°.
  - The middle bar fades and shrinks.
  - The bottom bar rotates −45°.
- Together they create a smooth close (✕) icon.

## Customization

You can easily customize:

- Bar color
- Background color
- Button size
- Animation duration
- Border radius
- Shadow effects

## Browser Support

Works in all modern browsers supporting CSS transforms and transitions.

## Preview

Click the circular menu button to toggle between the hamburger and close icons.
