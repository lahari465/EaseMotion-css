# 🕐 Clock Loader – Hand Spins Like Clock

> A classic, elegant loading animation where a clock hand spins like a clock, creating a timeless, sophisticated visual experience.

---

## 📖 Description

The **Clock Loader** component creates a classic, elegant loading animation where a clock hand spins like a clock. The component features a beautifully designed clock face with hour markers, numbers (12, 3, 6, 9), and a sleek hand that rotates continuously. The hand features a gradient color and a subtle arrow tip, with a center dot anchoring the design.

The component uses CSS animations with `transform: rotate()` to create the spinning hand effect. The clock face includes hour markers with prominent markers at 12, 3, 6, and 9, styled in blue for visual interest. The design is clean, minimalist, and reminiscent of classic analog clocks.

Perfect for loading states, waiting screens, and any interface where you want a sophisticated, timeless loading animation. The Clock Loader transforms a simple loading indicator into an elegant, refined visual experience.
