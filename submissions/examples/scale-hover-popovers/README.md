# Scale Hover Popover

A responsive CSS-only Scale Hover Popover built with HTML and CSS.

## Features

- Pure HTML & CSS
- Smooth scale animation
- Responsive design
- Keyboard accessible
- prefers-reduced-motion support
- No JavaScript required

## Folder

```
scale-hover-popover/
├── demo.html
├── style.css
└── README.md
```

## Usage

1. Open `demo.html`.
2. Hover or focus on the button.
3. The popover smoothly scales into view.
