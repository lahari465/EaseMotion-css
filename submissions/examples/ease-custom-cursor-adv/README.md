# 🎯 Custom Cursor – Circle Follows Mouse

> A smooth, modern custom cursor where a circle with a dot follows the mouse, creating an interactive, engaging user experience with hover and click feedback.

---

## 📖 Description

The **Custom Cursor** component replaces the default cursor with a smooth, animated circle that follows the mouse. The cursor consists of two elements: a small dot (the inner cursor) that follows the mouse immediately, and a larger ring (the outer cursor) that follows with a smooth, spring-like delay for a fluid, premium feel.

The cursor responds to interactions with hover and click feedback. On hover over interactive elements (buttons, links, etc.), the ring expands and changes color, while the dot grows slightly. On click, both elements contract with a satisfying pulse effect. The cursor is hidden on touch devices for a seamless mobile experience.

Perfect for creative portfolios, interactive websites, modern interfaces, and any application where you want to create a polished, engaging user experience. The Custom Cursor transforms a simple interaction into a premium, tactile experience.
