# CSS Bounce-Pulse Popover

A responsive popover component featuring a smooth bounce and pulse entrance animation built using pure HTML and CSS.

## Features

- Pure HTML & CSS
- Bounce-Pulse animation
- Responsive layout
- CSS Custom Properties
- Keyboard accessible
- Supports `prefers-reduced-motion`
