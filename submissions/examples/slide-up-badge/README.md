# CSS Slide-Up Badge

A responsive badge component with a smooth slide-up hover animation built using only HTML and CSS.

## Features

- Pure HTML & CSS
- Slide-up animation
- Responsive layout
- CSS Custom Properties
- Keyboard accessible
- Supports `prefers-reduced-motion`

## CSS Variables

```css
:root{
  --badge-bg:#2563eb;
  --badge-color:#fff;
  --radius:999px;
  --duration:.45s;
}
```
