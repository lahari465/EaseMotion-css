# Ease Kubernetes Container Probe Monitor

## Overview

A responsive Kubernetes workload diagnostics component for visualizing container liveness, readiness, and startup probe health.

## Features

- Liveness probe monitoring
- Readiness probe monitoring
- Startup probe monitoring
- Probe latency display
- Healthy, degraded, and unhealthy states
- Animated heartbeat indicators
- Failure threshold information
- Container diagnostic alerts
- Responsive layout
- Pure HTML and CSS

## Folder Structure

- `demo.html`
- `style.css`
- `README.md`

## Usage

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Why EaseMotion CSS?

This component provides a reusable Kubernetes diagnostic interface for understanding container health checks and identifying failing probes through animated visual states.
