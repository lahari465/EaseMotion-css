# CSS Shimmer Sweep Card Grid

A modern responsive portfolio card grid featuring a shimmer sweep animation using only HTML and CSS.

## Features

- Pure HTML & CSS
- Responsive Grid Layout
- Shimmer Sweep Hover Animation
- Smooth Transitions
- Mobile Friendly
- Accessible
- Supports prefers-reduced-motion
- No JavaScript Required

## Folder

```
shimmer-sweep-card-grid/
│── demo.html
│── style.css
└── README.md
```

## Usage

```html
<link rel="stylesheet" href="style.css">
```

Use the `.card-grid` container with multiple `.card` elements.

## Customization

Modify colors, animation duration, spacing, border radius and shadow directly in `style.css`.

## Accessibility

- Keyboard-friendly layout
- Reduced Motion support
- Responsive design
