Accordion (Dark Mode Aware)

Toggle your OS theme to see it switch automatically — no JS.

What is EaseMotion-css?

A collection of CSS animation examples contributed by the community.

How does dark mode work here?

CSS custom properties are redefined inside a prefers-color-scheme: dark media query, so the palette follows the user's OS setting automatically.

Does this need JavaScript?

No. The expand/collapse behavior uses the native <details>/<summary> elements, and theme switching is handled entirely by the browser via media queries.
